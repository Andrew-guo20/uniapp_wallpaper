## 阶段三：功能增强（4 天）

> 注意：阶段二（后台管理）不涉及 wallpaper/ 代码，wallpaper 从阶段一直接跳到阶段三。

### 3.1 收藏功能（1 天）

**预览页改造 `pages/preview/preview.vue`：**
- [ ] 底部操作栏新增 ❤️ 收藏按钮
- [ ] 已收藏状态红色高亮 + 点击取消收藏
- [ ] 乐观更新：先更新 UI 再调 API
- [ ] 未登录点击 → 弹起登录页

**API 层新增：**
- [ ] `apiAddFavorite({wallId})`
- [ ] `apiRemoveFavorite({wallId})`
- [ ] `apiToggleFavorite({wallId})`
- [ ] `apiIsFavorited({wallIds})` — 批量查询（列表页用）

**列表页改造 `pages/classlist/classlist.vue`：**
- [ ] 支持 `type=favorites` 参数
- [ ] 收藏模式下调用 `apiGetHistoryList({type:'favorite'})`
- [ ] 壁纸卡片右上角显示收藏小图标

**用户中心 `pages/user/user.vue`：**
- [ ] "我的收藏"入口 → 跳转 `classlist?type=favorites`

### 3.2 评论功能（1 天）

**预览页改造 `pages/preview/preview.vue`：**
- [ ] 底部操作栏 💬 评论按钮 → 弹出评论面板
- [ ] 评论面板组件：列表分页 + 输入框 + 发送按钮
- [ ] 每条评论：头像（默认）+ 时间 + 内容
- [ ] 长按自己的评论可删除

**API 层新增：**
- [ ] `apiAddComment({wallId, content})`
- [ ] `apiGetComments({wallId, pageNum, pageSize})`
- [ ] `apiDeleteComment({commentId})`

**交互细节：**
- [ ] 发送后清空输入框，评论插入列表顶部
- [ ] 乐观更新 + 失败回滚
- [ ] 空评论提示文案

### 3.3 用户投稿（0.5 天）

**新增页面 `pages/upload/upload.vue`：**

- [ ] 图片预览 + 重新选择

**API 层新增：**

**用户中心：**
- [ ] "我的投稿"入口 → 投稿列表（审核状态标签）
