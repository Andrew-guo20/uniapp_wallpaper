## 阶段一：用户体系（3 天）

> **目标**：微信一键登录，用户中心升级，deviceId 兼容过渡

### 1.1 新增登录页 `pages/login/login.vue`（1 天）

- [ ] 微信授权一键登录按钮（使用 `/frontend-design` 设计）
- [ ] 登录中 loading 动画
- [ ] 登录失败提示 + 重试
- [ ] 登录成功跳转来源页
- [ ] 游客模式关闭按钮（返回上一页不登录）

### 1.2 Token 管理 `utils/auth.js`（0.25 天）

- [ ] Token 存储/读取/清除（`uni.setStorageSync`）
- [ ] 登录状态判断 `isLoggedIn()`
- [ ] Token 过期检测

### 1.3 App.vue 全局登录态（0.25 天）

- [ ] `onLaunch` 检查 token
- [ ] 全局 `reactive` 用户状态（`isLogin`, `userInfo`）
- [ ] Token 过期静默刷新

### 1.4 用户中心改造 `pages/user/user.vue`（1 天）

- [ ] 已登录：微信头像 + 昵称展示（使用 `/frontend-design` 优化视觉）
- [ ] 未登录：占位头像 + "点击登录"引导
- [ ] 数据面板新增"我的收藏"计数
- [ ] 新增入口：我的收藏 / 我的投稿 / 我的评论
- [ ] 退出登录按钮（二次确认）

### 1.5 API 层扩展 `API/apis.js`（0.25 天）

- [ ] `apiLogin()` — 微信登录
- [ ] `apiGetUserProfile()` — 获取用户信息
- [ ] 更新 `apiGetUserInfo()` 返回值增加 `favoriteSize`

### 1.6 兼容性过渡（0.25 天）

- [ ] 未登录评分/下载 → 弹登录引导（不阻断）
- [ ] 搜索历史本地 → 云端（登录后迁移）
- [ ] deviceId 相关调用保持不变（后端兼容）

## 阶段二：互动功能（3.5 天）

> **目标**：收藏、评论、个性化推荐、搜索增强四大互动能力

### 2.1 壁纸收藏（1 天）

**预览页改造 `pages/preview/preview.vue`：**
- [ ] 底部操作栏新增 ❤️ 收藏按钮（`/frontend-design` 设计交互）
- [ ] 已收藏红色填充 + 点击取消（乐观更新）
- [ ] 未登录 → 弹登录引导

**API 层新增：**
- [ ] `apiToggleFavorite({wallId})` — 智能切换
- [ ] `apiIsFavorited({wallIds})` — 批量查询

**列表页改造 `pages/classlist/classlist.vue`：**
- [ ] 支持 `type=favorites` 参数展示收藏列表
- [ ] 壁纸卡片右上角收藏小图标

**用户中心 `pages/user/user.vue`：**
- [ ] "我的收藏"入口 → 跳转收藏列表

### 2.2 壁纸评论（1 天）

**预览页改造 `pages/preview/preview.vue`：**
- [ ] 底部 💬 评论按钮 → 弹出评论面板（`/frontend-design` 设计面板样式）
- [ ] 评论列表分页滚动
- [ ] 底部输入框 + 发送按钮
- [ ] 每条评论：默认头像 + 昵称 + 时间 + 内容
- [ ] 长按自己评论可删除

**API 层新增：**
- [ ] `apiAddComment({wallId, content})`
- [ ] `apiGetComments({wallId, pageNum, pageSize})`
- [ ] `apiDeleteComment({commentId})`

### 2.3 个性化推荐（0.5 天）

**首页改造 `pages/index/index.vue`：**
- [ ] "每日推荐"区域改为"为你推荐"
- [ ] 有登录+历史 → `apiGetRecommendWalls()` 个性化推荐
- [ ] 无历史 → `apiGetDayRandom()` 热门兜底

**API 层新增：**
- [ ] `apiGetRecommendWalls()`

### 2.4 搜索增强（0.5 天）

**搜索页改造 `pages/search/search.vue`：**
- [ ] 搜索历史云端同步
- [ ] 热门搜索词标签区
- [ ] 搜索结果收藏状态显示

**API 层新增：**
- [ ] `apiGetHotSearchKeywords()`
- [ ] `apiGetSearchHistory()`
- [ ] `apiClearSearchHistory()`

### 2.5 收藏列表页 `pages/favorites/favorites.vue`（0.5 天）

- [ ] 收藏壁纸网格展示
- [ ] 长按取消收藏
