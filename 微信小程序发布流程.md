## 一、微信公众平台配置

### 1.1 登录微信公众平台

访问 [https://mp.weixin.qq.com](https://mp.weixin.qq.com)，使用小程序账号登录。

### 1.2 配置服务器域名（必须）

路径：**开发 → 开发管理 → 开发设置 → 服务器域名**

点击"开始配置"，添加以下域名：

| 类型 | 域名 | 说明 |
|------|------|------|
| request 合法域名 | `https://env-00jy6ar8nt4o.dev-hz.cloudbasefunction.cn` | 云对象 URL 化域名 |
| uploadFile 合法域名 | `https://env-00jy6ar8nt4o.normal.cloudstatic.cn` | 云存储上传 |
| downloadFile 合法域名 | `https://env-00jy6ar8nt4o.normal.cloudstatic.cn` | 云存储下载（壁纸图片） |
| downloadFile 合法域名 | `https://picsum.photos` | 分类封面图（临时占位） |

> **注意：** 域名配置后需要一段时间生效。如果后续替换了分类封面图（不再依赖 picsum.photos），可移除该域名。

### 1.3 关闭域名校验（仅开发调试阶段）

如果暂时无法配置域名，可在 **微信开发者工具** 中：

详情 → 本地设置 → 勾选"**不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**"

> 仅限开发调试！正式发布前必须在微信公众平台配置好域名。

### 1.4 配置业务域名（如有 web-view）

本项目不使用 web-view，可跳过。

## 二、uni-app 项目配置

### 2.1 确认 manifest.json

`wallpaper/manifest.json` 中微信小程序配置：

```json
"mp-weixin": {
    "appid": "wxedf7c9be266ef39c",
    "setting": {
        "urlCheck": false
    },
    "usingComponents": true
}
```

- `appid` — 确认为你自己的小程序 AppID
- `urlCheck: false` — 开发阶段关闭域名检查

### 2.2 确认 uniCloud 配置

`manifest.json` 中应包含：

```json
"uniCloud": {
    "mode": "online"
}
```

确保云对象已通过 HBuilderX 上传部署到服务空间。

---

## 三、云对象重新部署（发布前必做）

在 HBuilderX 中打开 `wallpaper` 项目：

1. 展开 `uniCloud-alipay/cloudfunctions/wallpaper`
2. 右键 → **上传部署**
3. 等待部署完成

> 发布前务必部署一次，确保云端是最新代码。

---

## 四、构建微信小程序

### 4.1 在 HBuilderX 中发行

1. 顶部菜单：**发行 → 小程序-微信**
2. 等待编译完成
3. 编译产物在 `wallpaper/unpackage/dist/build/mp-weixin/`

### 4.2 在微信开发者工具中导入

1. 打开**微信开发者工具**
2. 选择"**导入项目**"
3. 项目目录：选择 `wallpaper/unpackage/dist/build/mp-weixin/`
4. AppID：`wxedf7c9be266ef39c`
5. 点击"确定"

### 4.3 测试

在微信开发者工具中逐页测试：

| 页面 | 检查项 |
|------|--------|
| 首页 | 轮播图、每日推荐、分类精选、公告滚动 |
| 分类页 | 10 个分类卡片展示 |
| 列表页 | 壁纸网格、触底加载更多 |
| 预览页 | 左右滑动、评分、下载保存、壁纸信息 |
| 搜索页 | 关键词搜索、历史记录、热门搜索 |
| 公告详情 | 富文本渲染 |
| 用户中心 | 下载/评分数、我的下载、我的评分 |

## 五、上传代码

### 5.1 在微信开发者工具中上传

1. 确认所有页面测试通过
2. 点击右上角"**上传**"
3. 填写版本号（如 `1.0.0`）和版本描述（如"初始版本，包含 10 个分类 200+ 壁纸"）
4. 点击"上传"

### 5.2 版本号规范

建议使用语义化版本：
- 当前版本：`1.0.0`
- 后续迭代：`1.0.1`（修 bug）、`1.1.0`（加功能）

---

## 六、微信公众平台提交审核

### 6.1 进入版本管理

登录微信公众平台 → **管理 → 版本管理**

### 6.2 提交审核

1. 在"开发版本"中找到刚上传的版本
2. 点击"**提交审核**"
3. 填写审核信息：
   - **服务类目**：工具 → 图片/壁纸
   - **标签**：壁纸、高清壁纸、手机壁纸
   - **测试账号**：如有登录功能需提供，当前无登录可跳过

### 6.3 审核时间

通常 1-7 个工作日，节假日可能更长。

---

## 七、发布上线

1. 审核通过后，在"版本管理 → 审核版本"中点击"**发布**"
2. 可选择**灰度发布**（部分用户）或**全量发布**
3. 发布后用户即可在微信中搜索到小程序

---

## 八、后续更新流程

每次更新代码后的发布流程：

```
1. 修改代码

3. HBuilderX 发行 → 小程序-微信
```

## 九、常见问题

### Q: 小程序打开白屏？
- 检查微信公众平台"服务器域名"是否配置了云对象和云存储域名
- 或在开发者工具中勾选"不校验合法域名"临时测试

### Q: 图片不显示？
- 检查 downloadFile 合法域名是否包含云存储域名
- 确认云存储权限为"公共读"

### Q: 评分/下载功能报错？
- 确认云对象已部署
- 检查 request 合法域名是否包含云对象域名

### Q: 搜索无结果？
- 确认搜索关键词匹配的壁纸存在
- 当前搜索支持：描述、标签、发布者、分类名称

### Q: 如何预览未发布版本？
- 微信公众平台 → 版本管理 → 开发版本 → 设为"体验版"
- 扫码即可在小程序中体验
